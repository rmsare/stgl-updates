NASA stereo pipeline installed

Published: Fri 12 May 2017
By Robert Sare

In Updates.

tags: updates

The NASA stereo pipeline is now installed on hilley-agisoft-linux. It is located in /opt/stereo_pipeline. The tools for the pipeline are in the bin/ subdirectory. The current version is 2.5.3.

Check out their documentation for more details.
