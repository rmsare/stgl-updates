Updates to Agisoft Photoscan

Published: Sun 05 February 2017
By Robert Sare

In Updates.

tags: updates photoscan

Agisoft Photoscan has been upgraded to the latest stable release, version 1.3.0 (build 3772). It has been activated for all users and will execute by default using photoscan.sh.

The following versions are installed on hilley-agisoft-linux:

Version Build Location
1.3.0 3772 /opt/photoscan-pro
1.3.0 3520 /opt/photoscan-pro_1_3_0-PRERELEASE
1.2.6 2834 /opt/photoscan-pro_1_2_6

Please note that projects created with 1.3.0 cannot be read by previous versions.
