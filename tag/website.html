This blog is for tracking updates to the hilley-agisoft-linux machine. The URL is https://rmsare.github.io/stgl-updates.

How to use this site

Feel free to write posts in /home/www/stgl-updates/content. You can write them in Markdown or RST. See .md files in that directory for examples of Markdown posts. Everyone in the www group has write access.

All pages are published throguh Github. Let me know if you want to be added as a collaborator to publish your changes.

Create your post by typing:

cd /home/www/stgl-updates
pelican content/ -o output -s pelicanconf.py
cd output/
git push origin gh-pages

Github repository

All content is stored in a github repository in the gh-pages branch.

Proudly powered by Pelican, which takes great advantage of Python.

The theme is by Smashing Magazine, thanks!
