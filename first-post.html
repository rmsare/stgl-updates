This blog is for tracking updates to the hilley-agisoft-linux machine.

How to use this site

Feel free to write posts to /home/www/stgl-updates/content. You can write them in Markdown or RST. See .md files in that directory for examples of Markdown posts.

Create your post by typing:

cd /home/www/stgl-updates
pelican content/ -o output -s pelicanconf.py
ghp-import output
git push origin gh-pages

Github repository

All content is stored in a github repository in the gh-pages branch.

Proudly powered by Pelican, which takes great advantage of Python.

The theme is by Smashing Magazine, thanks!
